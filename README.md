# Hotel Langhe & Monferrato

## Avvio rapido

```bash
npm install
npm run dev
```

Apri http://localhost:3000

## Pagine disponibili
- `/` → Homepage
- `/camere` → Tutte le camere
- `/camere/standard` → Camera Standard
- `/camere/tripla` → Camera Tripla
- `/camere/quadrupla` → Camera Quadrupla
- `/camere/suite` → Suite Deluxe

## Modificare i dati delle camere
Apri `lib/roomsData.js` e modifica testi, foto e prezzi.

## Deploy su Vercel
1. Carica su GitHub
2. Importa su vercel.com
3. Clicca Deploy
